# Practicalode-3 🐍

פרויקט לימודי: פרויקט Python בסיסי עם מבנה של אפליקציה/סקריפט + תבניות (templates)

## 📄 תיאור  
פרויקט זה נועד להדגים — במסגרת קורס / תרגול — עבודה עם קוד Python, תיקיית תבניות (`templates/`) וארגון קוד.  
הפרויקט מכיל קובץ `main.py` ותיקיית `templates/`.  
מטרות עיקריות:  
- להבין מבנה פרויקט Python בסיסי  
- להתנסות ב-Git — ניהול גרסה, `.gitignore`, README ועוד  
- לבנות תחנה ראשונה לקראת הרחבה עתידית (אם תרצי להוסיף ממשק, עיבוד נתונים, או רכיבי web)
